<br/><br/>

<h3>Instructions</h3>
1) cd to where you save projects<br/>
2) git clone *repo url.git* *folder-name*<br/>
3) open folder in text editor<br/>
4) git remote set-url origin *new repo url*<br/>
5) npm install --only=dev<br/>
6) gulp watch<br/>
7) GO! GO! GO!<br/>
