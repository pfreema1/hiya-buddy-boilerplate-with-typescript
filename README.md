<h3>Instructions</h3>
1) cd to where you save projects
2) git clone *repo url.git* *folder-name*
3) open folder in text editor
4) git remote set-url origin *new repo url*
5) npm install --only=dev
6) gulp watch
7) GO! GO! GO!
